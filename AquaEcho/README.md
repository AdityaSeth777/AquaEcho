# AquaEcho

AquaEcho is a sound-based navigation system designed to assist visually impaired and competitive swimmers with real-time lane guidance using haptic feedback (Apple Watch) and spatial audio cues (AirPods).

## Features

- Real-time lane position tracking using CoreMotion
- Spatial audio feedback via AirPods
- Haptic feedback through Apple Watch
- AI-powered lap detection
- HealthKit integration for swim analytics

## Requirements

- iOS 16.0+
- watchOS 9.0+
- Xcode 14.0+
- Swift 5.7+
- AirPods Pro (recommended for spatial audio)
- Apple Watch Series 4 or later

## Installation

1. Open the project in Xcode or Swift Playgrounds 4.5+
2. Build and run on your iOS device
3. Ensure you have paired AirPods and Apple Watch

## Usage

1. Grant required permissions (HealthKit, Motion)
2. Calibrate the system before your swim
3. Start your swim session
4. Receive real-time feedback through audio and haptics
